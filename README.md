# End-to-End Horticulture Analytics

This project performs **end-to-end data analytics on horticulture production data (2020–2023)** to uncover insights about crop cultivation, production trends, and geographical distribution across districts and mandals.

The workflow begins with **collecting raw agricultural datasets**, followed by **data cleaning and preprocessing in Excel** to remove inconsistencies, missing values, and irrelevant fields. The cleaned data is then **normalized and stored in a MySQL database**, enabling efficient querying and structured data management.

Using **Python (Pandas, NumPy, Matplotlib, and Seaborn)**, the project performs exploratory data analysis and generates visualizations to identify patterns such as:

* Yearly production and cultivation trends
* District-wise production comparison
* Crop type distribution
* Area vs production correlation

Finally, an **interactive dashboard in Power BI** presents key insights through charts and metrics, helping stakeholders easily understand agricultural productivity patterns.

## Tech Stack

* Excel – Data Cleaning & Preprocessing
* MySQL – Database Design & SQL Queries
* Python – Data Analysis & Visualization
* Pandas, NumPy – Data Manipulation
* Matplotlib, Seaborn – Visualization
* Power BI – Interactive Dashboard

## Key Insights

* Analysis of horticulture **area and production trends from 2020–2023**
* Identification of **top producing districts and crop types**
* Comparison of **cultivated area vs production output**
* Visualization of **geographical crop distribution**

## Outcome

This project demonstrates a **complete data analytics pipeline**, transforming raw agricultural data into meaningful insights through data cleaning, database management, analysis, and visualization.
